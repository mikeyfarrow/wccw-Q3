ALTER SEQUENCE

Change the next value of a sequence.

Syntax:

   ALTER SEQUENCE [schema.]sequence_name option(s)

Options:
      INCREMENT BY int
      MAXVALUE int | NOMAXVALUE
      MINVALUE int | NOMINVALUE
      CYCLE | NOCYCLE
      CACHE int | NOCACHE
      ORDER | NOORDER
